# KUN

## 1. Summary
This project, called KUN, is an application that delivers academic notice, departmental notice, professor information, and departmental office information from Konkuk University.
If you add your department at bookmark, you can easliy access departmental notice, professor information, departmental office information, etc.

## 2. Operating Environment
- Network must be connected because it needs to use Jsoup API for crawling data from the Konkuk University homepage.
- It's an Android application that doesn't work in an iOS environment.
- It ensures normal opration from Android 6.0(Marshmallow) to Anroid 10.0(Q).
- It requires very small storage space because it uses SQLite Database to save language option, boomark informaion in device.
